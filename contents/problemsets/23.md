# Soal 1–3<br>Permainan Segitiga

Pada lomba *Informatics Challenge*, para berang-berang bermain permainan di lapangan. Di lapangan terdapat 6 tiang kayu berjejer dengan panjang sebagai berikut. $$ 2, 4, 5, 7, 8, 10. $$

Seekor berang-berang boleh memilih 3 tiang berbeda untuk membuat sebuah segitiga. Jika berang-berang memilih tiang dengan panjang $i$, $j$, dan $k$ dengan $i \le j \le k$, maka skor segitiga yang akan ia dapatkan adalah $$ \mathrm{skor} = |i-j| + |j-k|. $$

Setelah membuat satu segitiga, tiang-tiang tersebut tidak boleh dipakai lagi. Berang-berang boleh membuat beberapa segitiga selama masih ada tiang yang tersisa. Skor yang didapatkan adalah total skor dari keseluruhan segitiga yang dibuat.

?? Berapa skor maksimum yang bisa diperoleh dengan 6 tiang kayu tersebut?

?? Jika saat ini tiang kayu sudah banyak, yakni sebanyak 12 tiang kayu dengan panjangnya adalah $$ 9, 10, 11, 14, 15, 16, 17, 20, 21, 22, 24, 25. $$ Berapa skor maksimum yang dapat diperoleh?

?? Dari 12 tiang kayu pada soal sebelumnya, ada berapa banyaknya cara memilih sehingga berang-berang mendapatkan nilai yang paling maksimal?

# Soal 4–6<br>A.a.

Perhatikan potongan kode berikut.

```cpp
int A(int a) {
	if (a < 0) {
		cout << "1";
		return A(-a);
	}

	if (a <= 1) {
		cout << (1 - a);
		return 0;
	}

	int q = a / 2;
	int r = a - 2 * q;

	A(q);

	cout << (1 - r);
	return 0;
}
```

?? Apa keluaran dari pemanggilan $A(10)$?

?? Pada pemanggilan $A(2026)$, berapa kali $A(q)$ dijalankan?

?? Untuk pemanggilan $A(M)$ mengeluarkan bilangan 8 digit, berapa nilai terkecil $M$ yang mungkin?

# Soal 7–9<br>*Cek Aja Dulu!* 

Perhatikan potongan kode berikut.

```cpp
void cek(bool a, bool b, bool c, bool d) {
	if(((a && !b) || c) && !((c && b) || (d && !a))) {
		cout << "kasus 1";
	} else if ((!c && b) || (a && b && !c && d)) {
		cout << "kasus 2";
	} else if ((!a && !b && (c || d)) && (c && d)) {
		cout << "kasus 3";
	} else {
		cout << "kasus 4";
	}
}
```

Untuk setiap nilai kebenaran bernilai benar, tuliskan TRUE. Jika nilai kebenaran bernilai salah, tuliskan FALSE.

?? Diketahui program mencetak `kasus 3`.

- Apa nilai kebenaran dari “$a$ AND $b$”?
- Apa nilai kebenaran dari “$c$ AND $d$”?

?? Diketahui program mencetak `kasus 1` dan $a = \mathrm{TRUE}$.

- Apa nilai kebenaran dari “$a$ AND NOT $b$”?
- <span style="color: red;">Apa nilai kebenaran dari “NOT ($c$ AND $d$)”?$^\dagger$</span>

^: $^\dagger$Tidak bisa ditentukan, soal keliru.
