Soal 1–3
Robot Penjaga Jalur

Robot cerdas sedang memeriksa sebuah jalur lurus yang terdiri dari beberapa tiang sinyal. Setiap tiang memiliki tinggi sinyal tertentu.

Sebuah tiang $T$ memiliki memiliki tinggi $T_h$ dan terletak pada posisi ke-$T_p$ dari kiri. Robot menganggap tiang sebagai koordinat $(T_h, T_p)$. Sebagai contoh jika tinggi sinyal yang tercatat adalah 2, 4, 1, dan 3, maka robot mencatat $(2, 1)$, $(4, 2)$, $(1, 3)$, dan $(3, 4)$.

Robot mengukur jarak antara dua titik menggunakan jarak Manhattan $d(A, B) = |A_h - B_h| + |A_p - B_p|$.

Robot mengatakan tiga tiang membentuk pola bermasalah jika $d(A, C) = d(A, B) + d(B, C)$ dengan $A_p < B_p < C_p$.

Robot dapat memilih beberapa tiang yang berurutan. Pilihan seperti ini disebut subjalur. Suatu subjalur disebut aman jika tidak ada tiga tiang di dalamnya yang membentuk pola bermasalah. Dengan jelas kita tahu bahwa subjalur dengan 1 atau 2 tiang selalu aman.

Jika terdapat 10 tiang, berapa banyak subjalur yang dapat dibuat?

Dengan nilai tinggi sinyal sebelumnya (yakni 2, 4, 1, dan 3), berapa banyak subjalur aman yang ada?

Jika terdapat 6 buah tiang sinyal dengan tinggi masing-masing adalah 6, 9, 1, 9, 6, dan 1, berapa banyak subjalur aman yang ada?

Soal 4–6
Inilah Sang Bintang

Perhatikan potongan kode berikut.

int n = 1;
int i = 1;

while (i <= n) {
	int j = 1;

	while (j < n) {
		cout << '*' << endl;
		j = j * 10;
	}

	i = i * 2;
}

Berapa kali karakter * yang dikeluarkan pada potongan program tersebut?

Jika $n$ diubah menjadi 1000, berapa kali karakter * yang dikeluarkan?

Berapa nilai $n$ terkecil sehingga karakter * tercetak setidaknya 100 kali?
