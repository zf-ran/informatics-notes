Soal 1–3
Pesta Bambu

Di sebuah hutan bambu di Tiongkok saat perayaan Imlek, seekor panda lucu ingin menyusun bambu-bambu favoritnya menjadi segitiga keberuntungan. Bambu favoritnya ini berbeda-beda warna dengan panjang berbeda-beda.

Sebuah kelompok bambu tersebut disebut dapat dibuat menjadi segitiga keberuntungan jika tiga bambu di dalam kelompok itu bisa membentuk segitiga. Artinya, untuk tiga panjang $(x, y, z)$ dengan nilai yang terbesar $z$ memenuhi $x + y > z$.

Panda lucu ini ingin membagi deretan bambu menjadi beberapa kelompok segitiga keberuntungan.

Panda lucu memiliki bambu dengan panjang 5, 1, 2, 9, 25, dan 14. Apakah panda lucu dapat membuat setidaknya satu segitiga keberuntungan?

Pada musim kering tiba, bambu-bambu panda semakin sedikit sehingga hanya tersisa 3 buah bambu favorit dengan panjang 17, 8, dan $x$. Jika $a$ adalah nilai terkecil yang mungkin untuk $x$ dan $b$ adalah nilai terbesar yang mungkin untuk $x$, tentukan nilai dari $a + b$.

Pada musim bambu tiba, panda punya lebih banyak bambu. Sekarang ia mempunyai beberapa bambu dengan panjang 4, 6, 7, 8, 10, 12, 14, 16, 19, 20, 21, dan 30. Tentukan ada berapa banyak segitiga keberuntungan yang dapat dibuat.

Soal 4–6
Takjil Spesial

Di meja masjid ada takjil berbentuk grid $n \times m$ ($n$ barus dan $m$ kolom). Ada satu takjil spesial yang awalnya ada di petak tertentu $(a, b)$: baris ke-$a$, kolom ke-$b$.

Dua panitia bermain. Ustaz Falsé ingin permainan berakhir secepatnya. Sedangkan Ustaz Maaf ingin permainan berakhir selama mungkin.

Pada setiap ronde:

  • Ustaz Falsé dapat memotong grid menjadi dua bagian, secara horizontal atau vertikal. Meja menjadi dua bagian: bagian yang tidak memiliki takjil spesial bisa langsung dibagikan kepada jemaah.
  • Setelah dipotong, tentunya ukuran grid akan berubah, dan posisi takjil spesial akan berubah relatif dengan grid baru.
  • Ustaz Maaf dapat menukar posisi takjil spesial dan takjil biasa.

Permainan akan selesai saat ukuran grid menjadi $1 \times 1$ di mana hanya takjil spesial yang tersisa.

Terdapat grid dengan ukuran $6 \times 8$ dan posisi takjil spesial pada petak $(3, 4)$. Jika Ustaz Falsé memotong grid secara horizontal di antara baris ke-3 dan ke-4. Berapa banyak takjil biasa yang tersisa?

Terdapat grid dengan ukuran $3 \times 3$, dengan posisi takjil spesial berada pada petak $(2, 2)$. Berapa ronde permainan akan berjalan jika Ustaz Falsé dan Ustaz Maaf bermain secara optimal?

Terdapat grid dengan ukuran $22 \times 99$, dengan posisi takjil spesial pada petak $(20, 70)$. Berapa ronde permainan akan berjalan jika Ustaz Falsé dan Ustaz Maaf bermain secara optimal?

Soal 7–8
Dua Perulangan

Perhatikan kode berikut.

#include <bits/stdc++.h>
using namespace std;

int main() {
	int n = 5;
	int x[n];

	// Perulangan pertama.
	for (int i = 0; i <= n; i++)
		x[i - 1] = (2 * n) / i;

	// Perulangan kedua.
	for (int i = n - 1; i >= 1; i--)
		x[n - 1] = x[n - 1] + x[i - 1];

	cout << x[n - 1] << endl;
	return 0;
}
