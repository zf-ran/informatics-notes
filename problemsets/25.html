Soal 1–3
Barisan Indah

Suatu barisan disebut indah jika memenuhi semua syarat berikut.

  1. Setidaknya memiliki 3 elemen.
  2. Untuk setiap elemen kecuali yang pertama, terdapat suatu elemen di sebelah kiri (tidak harus berurutan) yang nilainya lebih kecil.
  3. Untuk setiap elemen kecuali yang terakhir, terdapat suatu elemen di sebelah kanan (tidak harus berurutan) yang nilainya lebih besar.

Sebagai contoh, $[1, 4, 2, 4, 7]$ dan $[1, 2, 4, 8]$ adalah barisan indah. Sedangkan $[1, 2]$, $[2, 2, 4]$, dan $[1, 3, 5, 3]$ bukanlah barisan indah.

Ingat bahwa subsekuens adalah barisan yang diperoleh dengan menghapus beberapa elemen dari barisan asli tanpa mengubah urutan elemen yang tersisa.

Barisan $[3, 1, 3, 2, 2, 1, 2]$ memiliki subsekuens yang merupakan barisan indah.

Ada berapa banyak subsekuens dari barisan $[3, 2, 1, 2, 2, 1, 3]$ yang merupakan barisan yang indah?

Ada berapa banyak subsekuens dari barisan $[1, 2, 3, 2, 1, 3, 2, 2, 3]$ yang merupakan barisan yang indah?

Soal 4–6
K.I.M.

Perhatikan potongan kode berikut.

int kunyah(int a, int b) {
	if (a <= 1 || a <= 1)
		return 0;

	int c = 0, d = 0;

	while (a > b) {
		a -= b;
		c++;
		d += b;
	}

	cout << c << "," << d << endl;
	kunyah(max(c, d), min(c, d));

	return a;
}

int iseng(int a, int b) {
	int x = kunyah(a, b);
	cout << x << "," << b << endl;

	return x ^ b;
}

int misterius(int a, int b) {
	int p = iseng(a, b);
	int q = kunyah(p, b);

	return q & p;
}

Tentukan jumlah angka-angka pada output dari pemanggilan $\mathtt{kunyah}(14, 3)$.

Tentukan hasil dari pemanggilan $\mathtt{iseng}(5, 2)$.

Ada berapa banyak karakter 1 yang dikeluarkan pada pemanggilan $\mathtt{misterius}(17,5)$.
